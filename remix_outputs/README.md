# Remix Outputs

This directory is where your generated remixes and DJ sets will be saved.

*   **Single Mashups:** Saved as `output_filename.mp3`.
*   **DJ Sets:** Saved as `ai_dj_set_<style>_<timestamp>.mp3`.
*   **Continuous Mixes:** Saved in dated subdirectories (e.g., `2025-11-30/`).

*Note: Audio files are ignored by git to keep the repository light.*
